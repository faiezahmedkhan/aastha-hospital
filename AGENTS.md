# Automatic Deployment
Whenever you successfully complete a task and verify your changes, you MUST automatically commit the changes using `git add .` and `git commit -m "your message"`, and then push to the origin using `git push`. This ensures the code automatically deploys to Vercel without the user having to explicitly ask every time.
